# Studio A7 // Branded 3D WebGL Multiplayer Wolf Bot Arena Game (v1.0.1)

Oficjalne repozytorium techniczne zespołu deweloperskiego agencji **Studio A7** ([www.studioa7.pl](https://www.studioa7.pl)). Projekt to interaktywna, wieloosobowa gra 3D (nowoczesny, zaawansowany remake klasycznego Snake'a), stworzona jako technologiczny pokaz możliwości (showcase) w zakresie optymalizacji Core Web Vitals, zaawansowanego renderowania grafiki w przeglądarce oraz technologii czasu rzeczywistego.

Jako agencja pokazujemy, że nowoczesne **pozycjonowanie stron** to nie tylko suchy tekst, ale też angażujące elementy interaktywne (grywalizacja), które drastycznie zwiększają czas spędzony na stronie (Dwell Time), co bezpośrednio przekłada się na autorytet domeny w Google.

---

## 🎮 O Projekcie (Wolf Bot Arena)

Gra została w pełni spersonalizowana pod markę Studio A7. Klasyczne kulek zastąpiono lśniącymi kryształami, a gracze zamiast zwykłych węży kontrolują geometryczne, trójwymiarowe roboty-wilki (Cyber-Wolves). 
Ten projekt to również nasz wewnętrzny deweloperski **przegląd od AI** najnowszych technologii frontendowych i backendowych. Testujemy tutaj unikalne rozwiązania z zakresu user experience, które wdrażamy później u naszych klientów, optymalizując m.in. rozwiązania pod kątem nowej ery wyszukiwarek i **seo dla AI**.

### 💻 Wykorzystany Stack Technologiczny (100% Real):
* **Frontend:** React 19 + Vite (błyskawiczne budowanie aplikacji) + Tailwind CSS (stylowanie UI).
* **Silnik 3D:** React Three Fiber (R3F) + Three.js – renderowanie zaawansowanej grafiki 3D bezpośrednio na warstwie HTML5 Canvas przy zachowaniu 60 FPS.
* **Backend Multiplayer:** Node.js + Express + Socket.io – autorytatywny serwer synchronizujący ruch wszystkich graczy w czasie rzeczywistym (protokół WebSocket).
